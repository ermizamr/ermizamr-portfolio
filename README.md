# Ermyas Misiker — Portfolio

Personal portfolio website for [Ermyas Misiker](https://www.ermizamr.tech) — Full-Stack Developer & Founder based in Addis Ababa, Ethiopia.

## Tech Stack

- React + TypeScript
- Tailwind CSS v4
- Vite
- Formspree (contact form)

## Sections

- **Hero** — Introduction with CTA buttons
- **About** — Short bio
- **Experience** — Work history & education
- **Skills** — Tech stack badges
- **Projects** — Featured projects with GitHub links
- **Contact** — Contact form via Formspree
